OK RECORDS es una tieda en la cual los clientes pueden adquirir CDS, viniles y Albumes de manera online. 
Se manenjan 15 generos musicales en los cuales se puede adquirir de nuestros articulos, 
se hace la entrega en donde el cliente elija y la ateción es personalizada.

Dependencias: 
-React te da la estructura para construir la UI.
-React-DOM se encarga de mostrar esa UI en el navegador.
-React-Router-Dom gestiona la navegación entre las distintas páginas o vistas de tu aplicación.
